About GLAND
-----------
GLAND is designed to extract high-level features from spatial transcriptomics data. It is compatible with various platforms, spanning from sequencing-based methods (10x Visium, Stereo-seq, and ST) to imaging-based technologies (MERFISH, osmFISH, BaristaSeq, and STARmap).
